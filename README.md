Сервис управления рассылками
📌 Описание проекта
Сервис позволяет пользователям создавать рассылки, управлять клиентами и сообщениями, а также отправлять письма вручную или по расписанию. В проекте реализованы роли пользователей, логирование попыток рассылки и кеширование с использованием Redis.

🚀 Функциональность
Основная часть (обязательная по ТЗ):
✅ CRUD для клиентов (email, ФИО, комментарий)
✅ CRUD для сообщений (тема, тело письма)
✅ CRUD для рассылок (время начала/окончания, статус, связь с сообщениями и клиентами)
✅ Отправка рассылок вручную через кнопку
✅ Логирование попыток отправки (успешно/не успешно, ответ сервера)
✅ Главная страница со статистикой (всего рассылок, активных, уникальных клиентов)
Дополнительная часть (расширение функциональности):
✅ Регистрация и аутентификация пользователей (через email)
✅ Вход/выход, подтверждение email через письмо
✅ Ограничение доступа: пользователь видит только свои данные, менеджер — все
✅ Роли: менеджеры и обычные пользователи
✅ Серверное кеширование главной страницы с использованием Redis
✅ Отправка рассылок через кастомную команду management send_mailings

📂 Установка
Клонировать репозиторий:
git clone https://github.com/yournameDevWebAppWithDjango.git
cd DevWebAppWithDjango
Создать и активировать виртуальное окружение:
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
Установить зависимости:
pip install -r requirements.txt
Настроить .env файл:
EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=
EMAIL_PORT=
EMAIL_USE_TLS=False
EMAIL_USE_SSL=True
EMAIL_HOST_USER=
EMAIL_HOST_PASSWORD=
DEFAULT_FROM_EMAIL=
Применить миграции:
python manage.py migrate
Создать суперпользователя:
python manage.py createsuperuser
Запустить Redis-сервер и затем запустить проект:
redis-server
python manage.py runserver
📨 Автоматическая отправка (опционально)
Запуск рассылки вручную из командной строки:

python manage.py send_campaigns
📌 Авторизация и роли
Пользователь может видеть и управлять только своими рассылками и клиентами
Менеджер видит все рассылки, но не может редактировать чужие
📊 Главная страница
Показывает общее количество рассылок, активных рассылок и уникальных клиентов. Кешируется через Redis на 2 минуты.
